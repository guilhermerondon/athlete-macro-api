# Athlete Macro API

Serviço focado em performance esportiva e saúde. Esta API processa e calcula as distribuições ideais de macronutrientes com base no metabolismo e nos objetivos de atletas.

## 🚀 Tecnologias e Arquitetura

- **Python & FastAPI**: Framework moderno e de extrema velocidade.
- **Pydantic**: Validação de esquemas e serialização de dados rigorosa.
- **PostgreSQL / SQLAlchemy**: Persistência do histórico de cálculos.
- **Rich Console**: Logging avançado no terminal para Developer Experience (DX).

## 🧠 Filosofia

A performance não aceita atalhos. Com 1,90m, 101kg e rotina de treino de 6 vezes na semana, a nutrição é a fundação. Esta API codifica a lógica por trás de um planejamento milimétrico de macros, alinhando disciplina atlética e engenharia de software precisa.

## 🛠️ Como Executar

```bash
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```
A documentação interativa fica em `/docs`.
